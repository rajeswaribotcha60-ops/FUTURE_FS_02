📌 Project Description

The Client Lead Management System (Mini CRM) is a simple and user-friendly web application designed to help manage client leads efficiently.

This application allows users to add and manage client information, track lead status, and maintain important follow-up details in one place.

The project demonstrates the use of HTML, CSS, and JavaScript to create an interactive and responsive CRM interface.

---

✨ Features

- ➕ Add new client leads
- 👤 Store client names and contact details
- 📧 Manage email information
- 📞 Manage phone numbers
- 📌 Track lead status
- 📅 Maintain follow-up information
- ✏️ Edit client details
- 🗑️ Delete client records
- 🔍 Search and manage leads easily
- 📱 Responsive and user-friendly interface

---

🛠️ Technologies Used

- HTML5 – For creating the structure of the application
- CSS3 – For styling and responsive design
- JavaScript – For interactive functionality and lead management

🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Open "index.html" in any modern web browser.
4. Use the CRM dashboard to add, edit, search, and delete client leads.
